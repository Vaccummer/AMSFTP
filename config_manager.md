# config manager

intro: a class to manage config.yaml(store host information for establish connections) and settings.yaml(store variables for internal use)

## Funtions

### Init

1. ✅list 打印所有config信息()
2. ✅src 获取config文件路径
3. ✅query 获取指定昵称的host信息
4. ✅delete 删除指定host
5. ✅add 增加一个host
6. ✅modify 修改指定host的信息
